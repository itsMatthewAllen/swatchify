# Change Log

All notable changes to the "color-variable-swatches" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [0.1.0] - 2026-02-27

### Added
- Initial public release of Swatchify
- Core variable resolver with cascading, scoping and media query support
- Color swatches for `var()` usages in CSS files
- Built-in test suite covering parsing, resolution, and integrations
- Activation on CSS, SCSS and LESS language modes (with only traditional CSS variable functioning currently)
